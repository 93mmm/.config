# Dependencies
- fish shell
- tmux
- neovim
- fzf, bat, ripgrep

# Installation
```sh
cd ~
mv .config .my-old-config
git clone https://github.com/93mmm/.config.git
```

## Configure tmux
```sh
git clone https://github.com/tmux-plugins/tpm ~/.config/tmux/plugins/tpm
```
press `prefix` + I (`prefix` stands for `ctrl` in my case) to install plugins
